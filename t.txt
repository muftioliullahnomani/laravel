Categories
POST /wp-json/ecom/v1/import/categories
Body (JSON):
json
[
  {"name": "T-Shirts", "slug": "tshirt"},
  {"name": "Shoes", "slug": "shoes"}
]
Products
POST /wp-json/ecom/v1/import/products
Body (JSON):
json
[
  {
    "title": "Blue Tee",
    "slug": "blue-tee",
    "content": "<p>100% cotton</p>",
    "price": 12.99,
    "categories": ["tshirt"],
    "image": "https://example.com/images/blue-tee.jpg"
  }
]
কাজ: নতুন হলে insert, slug মিলে গেলে update; _price মেটা সেভ; product_cat টার্ম সেট; image দিলে ফিচারড ইমেজ সাইডলোড।
Menus
POST /wp-json/ecom/v1/import/menus
Body (JSON):
json
[
  {
    "location": "primary",
    "items": [
      {"title": "Home", "url": "/", "align": "left"},
      {"title": "Cart", "url": "/cart", "align": "right"}
    ]
  }
]
কাজ: লোকেশন মেনু না থাকলে তৈরি + এসাইন; আগের আইটেম ক্লিয়ার করে নতুন আইটেম তৈরি; _menu_item_align সেট।
Styles
POST /wp-json/ecom/v1/options/styles
Body (JSON):
json
{
  "home_style": {"card":"rounded-lg shadow-sm", "image_ratio":"1/1"},
  "section_style": {"card":"rounded-md border", "image_ratio":"4/3"}
}
